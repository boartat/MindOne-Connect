# MindOne-Connect
MindOne Connect의 개인정보 처리방침입니다.

본 애플리케이션 "MindOne Connect"는 사용자의 개인정보를 수집, 저장, 또는 공유하지 않습니다.

키보드 입력을 포함한 모든 데이터는 사용자 기기 내에서만 처리되며, 외부 서버로 전송되지 않습니다.

또한 본 앱은 사용자 추적, 분석 도구, 또는 제3자 데이터 수집 서비스를 사용하지 않습니다.

개발자: 우리집은저지클럽

[Gless Privacy Policy]

The application "MindOne Connect" does not collect, store, or share any personal information.

All data, including keyboard input, is processed locally on the user's device and is not transmitted to any external servers.

This app does not use analytics, tracking, or any third-party services that collect user data.
